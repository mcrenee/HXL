# 项目筛选评估系统 - 使用指南

## 📋 系统概述

全新的结构化评分系统，基于**五大模块、15个维度**进行项目评估，总分范围为**1-5分**。

该系统优先考虑**资产安全性和现金流安全性**，而不是增长潜力。

---

## 🎯 评分结构

### 模块一：资产质量（权重30%）

评估底层资产本身是否具备长期经营价值，而不是依赖特定运营方。

**维度：**
1. **地段与需求强度**：是否位于核心商圈、机场、高铁站、医院或高密度办公区，客流是否稳定
2. **租约条件**：租金是否低于或接近市场水平，剩余租期是否足够长（理想为3年以上）
3. **房东质量**：房东是否为机场集团、国企或大型商业机构，稳定性是否高

### 模块二：单位经济模型（权重25%）

评估项目是否天然具备健康现金流。

**维度：**
4. **租金占收入比例**：租金占收入比例越低越好（低于40%为优秀，高于70%为危险）
5. **回本周期**：投资回本周期越短越好（12个月以内优秀，超过24个月风险较高）
6. **净现金流能力**：项目是否稳定产生正现金流

### 模块三：运营能力（权重20%）

评估运营方是否具备持续成功运营能力。

**维度：**
7. **历史项目表现**：是否有成功运营多个项目的记录
8. **扩张节奏健康度**：扩张是否稳健，是否依赖持续融资
9. **成本控制能力**：是否具备较强成本控制能力和运营效率

### 模块四：结构安全性（权重15%）

评估投资人在极端情况下是否具备资产控制权和资金保护。

**维度：**
10. **租约控制权**：投资人或SPV是否直接或间接控制租约
11. **现金流控制**：收入是否进入监管账户或可被投资人监控
12. **投资保护条款**：是否具备优先分配权、接管权或类似保护机制

### 模块五：扩张能力（权重10%）

评估业务是否具备规模化和长期增长能力。

**维度：**
13. **可复制性**：业务模式是否标准化，是否可复制到多个点位
14. **点位获取能力**：运营方是否具备持续获取优质点位能力
15. **抗风险能力**：如果收入下降30%，项目是否仍可存活

## 📊 评分方法

### 计算逻辑

1. 每个维度评分为 **1-5分**
   - 5分：极强/极安全
   - 4分：较强/安全
   - 3分：一般/中等
   - 2分：较弱/风险较高
   - 1分：极弱/极高风险

2. 每个模块取其内部维度的**平均分**

3. 最终总分 = 各模块平均分 × 对应权重之和

4. 总分范围：**1.0 - 5.0分**

### 示例计算

假设某项目评分如下：

**模块一（资产质量）：**
- 地段与需求强度：5分
- 租约条件：4分
- 房东质量：5分
- **模块一平均分：(5+4+5)/3 = 4.67分**

**模块二（单位经济模型）：**
- 租金占收入比例：4分
- 回本周期：5分
- 净现金流能力：4分
- **模块二平均分：(4+5+4)/3 = 4.33分**

**模块三（运营能力）：**
- 历史项目表现：4分
- 扩张节奏健康度：3分
- 成本控制能力：4分
- **模块三平均分：(4+3+4)/3 = 3.67分**

**模块四（结构安全性）：**
- 租约控制权：5分
- 现金流控制：4分
- 投资保护条款：4分
- **模块四平均分：(5+4+4)/3 = 4.33分**

**模块五（扩张能力）：**
- 可复制性：4分
- 点位获取能力：4分
- 抗风险能力：3分
- **模块五平均分：(4+4+3)/3 = 3.67分**

**最终总分：**
```
= 4.67 × 30% + 4.33 × 25% + 3.67 × 20% + 4.33 × 15% + 3.67 × 10%
= 1.40 + 1.08 + 0.73 + 0.65 + 0.37
= 4.23分
```

**投资建议：Strong Invest**

## ✅ 投资建议分级

| 总分范围 | 投资建议 | 说明 |
|---------|---------|------|
| ≥ 4.0分 | **Strong Invest** | 强烈推荐投资，资产安全性和现金流能力极强 |
| 3.5 - 3.9分 | **Invest** | 推荐投资，整体表现良好 |
| 3.0 - 3.4分 | **Cautious** | 谨慎投资，存在一定风险，需加强风控 |
| < 3.0分 | **Do Not Invest** | 不建议投资，不符合投资标准 |

## 📋 输出内容

系统将输出以下完整评估报告：

### 1. 总评分
- 显示格式：**X.X / 5.0**（保留一位小数）
- 投资建议标签：Strong Invest / Invest / Cautious / Do Not Invest

### 2. 各模块评分
- 五个模块的详细评分
- 每个模块显示权重和得分
- 使用不同颜色区分模块

### 3. 主要优势
- 自动识别评分≥4分的维度
- 最多显示3条
- 按评分从高到低排序

### 4. 主要风险
- 自动识别评分≤2分的维度
- 最多显示3条
- 按评分从低到高排序

### 5. 投资判断总结
- 3-5句话的综合评估
- 重点关注资产安全性和现金流
- 针对性的投资建议和风险提示

## 💡 评估要点

### 资产安全性优先

1. **关注模块一（资产质量30%）**
   - 核心商圈优于一般地段
   - 长期租约优于短期租约
   - 优质房东优于个人房东

2. **关注模块四（结构安全性15%）**
   - 投资人控制权
   - 现金流监管
   - 完善的保护条款

### 现金流安全性

1. **关注模块二（单位经济模型25%）**
   - 租金占比要低
   - 回本周期要短
   - 现金流要健康

2. **关注抗风险能力**
   - 收入下降30%仍可生存
   - 不依赖持续融资
   - 成本控制能力强

### 评分建议

- **保守评分**：对不确定的维度，宁可给低分
- **真实评分**：基于客观事实，不夸大
- **重点关注**：资产质量和结构安全性最重要
- **次要关注**：扩张能力权重最低，不必过度追求

## 📝 常见问题

### Q1: 如何判断一个项目是否值得投资？

A: 主要看三点：
1. 总分≥4.0分（Strong Invest）
2. 模块一（资产质量）≥4.0分
3. 模块四（结构安全性）≥3.5分

### Q2: 某个模块评分特别低怎么办？

A: 
- 如果是模块一或模块四评分<3.0，建议直接放弃
- 如果是模块三或模块五评分较低，可以考虑但需加强监管
- 如果是模块二评分<3.0，现金流风险太高，不建议投资

### Q3: 所有维度都是3分，可以投资吗？

A: 
- 总分会是3.0分（Cautious）
- 说明项目中规中矩，但无明显优势
- 建议谨慎投资，需要更强的风控和监管

### Q4: 如何快速判断项目质量？

A: 优先看这4个维度：
1. 地段与需求强度（维度1）
2. 净现金流能力（维度6）
3. 租约控制权（维度10）
4. 现金流控制（维度11）

如果这4个维度都≥4分，项目基本可靠。

## 🚀 最佳实践

### 投资决策流程

1. **初步筛选**：完成15维度评分，总分≥3.5分才考虑
2. **深入分析**：重点分析资产质量和结构安全性
3. **风险评估**：查看主要风险部分，评估可接受程度
4. **条款设计**：根据评分结果设计合理的投资条款
5. **监管方案**：根据弱项设计针对性的监管措施

### 不同评分对应策略

**Strong Invest (≥4.0分)**
- 可以给予较高估值
- 可以接受较低的收益率
- 可以适当放松监管

**Invest (3.5-3.9分)**
- 标准估值
- 标准收益率
- 正常监管频率

**Cautious (3.0-3.4分)**
- 保守估值
- 要求更高收益率
- 加强监管和风控

**Do Not Invest (<3.0分)**
- 直接放弃
- 或要求项目改进后重新评估
