# 超级加盟商行业投资决策平台

## 项目概述

这是一个专为**超级加盟商行业投资分析**设计的智能决策平台，基于滴灌通投资的实践经验和肆瓦商业的运营数据，提供全方位的行业分析、企业筛选和合同参考功能。

## ✨ 核心功能

### 1️⃣ 行业报告展示
- **3L+C 商业模型**：Labor（劳动力）、Land（高势能点位）、Leverage（杠杆）、Capital（资本）
- **核心数据指标**：120+门店、18%年化回报、15个月平均回本周期、70%毛利率
- **典型案例分析**：肆瓦-浦东机场T2项目详细数据
- **历史履约图表**：Chart.js 可视化展示投资回报数据

### 2️⃣ 被投企业智能筛选
- **8项评估标准**：
  - **行业通识标准**（权重30%）：
    1. 优质点位获取能力
    2. 收入与履约保障能力
    3. 拓展与整合能力
  - **专属分析标准**（权重70%）：
    4. 历史项目履约情况
    5. AI技术应用能力
    6. 品牌资源整合能力
    7. 资金使用效率
    8. 团队专业度与背景

- **智能评分系统**：
  - 总分200分，自动计算加权得分
  - 五级评级：优秀（170+）、良好（140+）、一般（110+）、不推荐（<110）
  - AI生成投资建议和风控要点（100字内）

### 3️⃣ 投资合同关键条款
- **核心商业条款**：RBF模式、投资金额、年化收益率、分成比例、联营期限
- **收入分成安排**：收入定义、分成计算、封顶机制、数据传输、分成付款
- **前置条件清单**：6项投资前置条件详细说明
- **风险控制措施**：招商保障、收入保障、数据核验、违约处理
- **各方权利义务**：甲乙丙丁四方责任明确划分
- **特殊约定事项**：分成前债务、税费承担、押金处理、预付卡收入

## 🚀 使用说明

### 本地运行
```bash
# 方法1：Python HTTP服务器
cd webapp
python3 -m http.server 3000

# 方法2：Node.js serve
cd webapp
npx serve

# 访问：http://localhost:3000
```

### 功能使用

#### 查看行业报告
1. 打开首页，查看**3L+C模型**和核心数据
2. 浏览**典型案例**了解投资详情
3. 查看**历史履约图表**分析回报数据

#### 企业筛选评估
1. 进入"企业筛选"模块
2. 填写8项评估标准（每项必选）
3. 点击"计算评分"查看结果
4. 系统自动生成：
   - 总分和分项得分
   - 投资评级（⭐1-5星）
   - 投资建议
   - 风控要点

#### 查阅合同条款
1. 进入"合同条款"模块
2. 查看6大类关键条款
3. 了解RBF投资模式详情
4. 参考风险控制措施
